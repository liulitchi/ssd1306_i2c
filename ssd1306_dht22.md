# Arduino | ssd1306 | DHT22

## 依赖

> Adafruit SSD1306 
> Adafruit Unified Sensor
> Adafruit GFX Library 
> DHT sensor library

以上四个库文件，均可在 Aeduino 内，`项目` --> `加载库` -->`管理库` 安装。

## 接线位置

以下为一个参考接线

|传感器接口|Arduino接口|显示屏接口|Arduino接口|
|---|---|---|---|
|DHT22 +|5V|VCC|5V|
|DHT22 -|GND|GND|GND|
|DHT22 out|D2|SCL|A5|
|-|-|SDA|A4|
